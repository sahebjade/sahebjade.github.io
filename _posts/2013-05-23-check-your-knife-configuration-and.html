Solution

Chef server uses FQDN for the redirection between the different components in this case was bookshelf and erchef. While install time the chef server we configure valid FQDN then we will face this issue..

So I created file called /etc/chef-server/chef-server.rb, and added below lines... 
server_name = "192.168.56.11"
api_fqdn server_name
nginx['url'] = "https://#{server_name}"
nginx['server_name'] = server_name
lb['fqdn'] = server_name
bookshelf['vip'] = server_name

Next I run chef-server-ctl reconfigure.

...I was able upload the cookbook to chef server now.
